# Contributing

## Flujo recomendado

```text
Issue
↓
Branch
↓
Implementación
↓
Pruebas
↓
Actualización del Handbook
↓
Pull Request
↓
Review
↓
Merge
```

## Antes de abrir Pull Request

Verificar:

- build correcto;
- pruebas mínimas;
- documentación actualizada;
- API actualizada si cambió contrato;
- Data Dictionary actualizado si cambió BD;
- Business Flow actualizado si cambió flujo;
- ADR actualizada si cambió arquitectura;
- Runbook actualizado si cambió operación.

## Regla

Una tarea no está terminada si deja conocimiento importante fuera del repositorio.
