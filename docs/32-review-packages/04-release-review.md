# Release Review Package

## Objetivo

Validar que una versión del Handbook o del software puede liberarse.

## Checklist

- ADR actualizadas.
- Domain actualizado.
- API actualizada.
- Business Flows actualizados.
- Data Dictionary actualizado.
- Runbooks actualizados.
- QA ejecutado.
- CHANGELOG actualizado.
- MkDocs build correcto.
