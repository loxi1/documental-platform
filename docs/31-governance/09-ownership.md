# Ownership

Todo documento relevante debe declarar:

```yaml
Owner:
Status:
Version:
Source of Truth:
Last Review:
Related:
```

## Owners oficiales

- Chief Architect: infraestructura, base de datos y operación.
- Domain Architect: motor documental, backend, API y data dictionary.
- Product Architect: producto, UI Foundation y frontend.
- Product Owner: roadmap y reglas funcionales.
- Knowledge Manager: coherencia documental y publicación.
