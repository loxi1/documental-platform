# Review Process

## Owners

| Dominio | Revisor |
|---|---|
| Infraestructura | Viejo Maestro / Chief Architect |
| Motor Documental | Maestro Sucesor I / Domain Architect |
| Producto | Maestro Sucesor II / Product Architect |
| Roadmap | Product Owner |
| Handbook | Knowledge Manager |

## Cuándo revisar

- Cambio de arquitectura.
- Cambio de regla de negocio.
- Cambio de API.
- Cambio de modelo de datos.
- Cambio de UX global.
- Cambio de despliegue.
