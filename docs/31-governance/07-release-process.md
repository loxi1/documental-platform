# Release Process

## Estados

- RC: release candidate.
- Final: versión aprobada.
- Patch: corrección menor.
- Minor: mejora compatible.
- Major: cambio estructural.

## Handbook v1.0

Se considera listo cuando:

- Source of Truth completo.
- Duplicados principales eliminados.
- MkDocs construye sin errores.
- Architecture Review Board aprueba.
