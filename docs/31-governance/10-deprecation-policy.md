# Deprecation Policy

## Regla

Los documentos no se eliminan directamente.

Cuando un documento queda obsoleto:

1. Se marca como Deprecated.
2. Se indica documento reemplazo.
3. Se mueve a archivo histórico cuando corresponda.
4. Se actualizan referencias cruzadas.

## Motivo

Conservar trazabilidad de decisiones y evolución del proyecto.
