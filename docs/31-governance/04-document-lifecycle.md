# Document Lifecycle

## Estados

```text
Draft → Review → Approved → Deprecated → Archived
```

## Draft

Documento en construcción.

## Review

Documento listo para revisión por owner.

## Approved

Documento aprobado como parte del Handbook.

## Deprecated

Documento reemplazado, pero conservado por trazabilidad.

## Archived

Documento histórico sin uso operativo.

## Regla

No eliminar documentos históricos sin registrar motivo.
