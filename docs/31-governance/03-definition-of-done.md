# Definition of Done

Una funcionalidad está terminada solo si cumple:

- Código implementado.
- Build ejecutado.
- Validación funcional realizada.
- Pruebas mínimas ejecutadas.
- Documentación actualizada.
- ADR actualizada si cambia arquitectura.
- Domain actualizado si cambia negocio.
- API actualizada si cambia contrato.
- Data Dictionary actualizado si cambia BD.
- Business Flow actualizado si cambia flujo.
- Runbook actualizado si cambia operación.
- Product Guidelines actualizadas si cambia UX.
