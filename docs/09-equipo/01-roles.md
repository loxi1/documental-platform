# Roles del Proyecto

## Objetivo

Definir responsabilidades oficiales dentro de Documental Platform.

## Roles

| Rol | Responsabilidad principal |
|---|---|
| Viejo Maestro | Arquitectura, infraestructura, baseline, deploy |
| Maestro Sucesor I | Motor Documental, backend, OCR, APIs |
| Maestro Sucesor II | Producto, frontend, UI Foundation, UX |
| Product Owner | Reglas de negocio, QA funcional, roadmap, integración |

## Regla

Cada rol debe respetar su dominio para evitar conflictos entre sprints.
