# Motor Documental

El Motor Documental es una capacidad compartida de la plataforma.

No pertenece exclusivamente al módulo Gestión Documental.

Será reutilizado por:

- Gestión Documental
- Caja Chica
- Rendiciones
- RRHH
- Portal de Proveedores

---

## Principios

- Documento lógico.
- Archivo físico.
- OCR propone.
- Backend dispone.
- Nunca sobrescribir.
- Siempre versionar.

---

## Componentes

- Confirmación
- OCR
- Versionado
- Clave Documental
- Preview
- Enriquecimiento por RUC
- Duplicados
