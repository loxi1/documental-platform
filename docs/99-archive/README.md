# Archive

Esta carpeta conserva documentos reemplazados, duplicados o históricos.

## Regla

No se eliminan decisiones antiguas sin trazabilidad.

Cuando un documento queda obsoleto:

1. Se mueve a `99-archive/`.
2. Se mantiene su contenido.
3. Se actualizan las referencias al documento oficial.
4. Se registra el cambio en `CHANGELOG.md`.

## No usar como fuente oficial

Los documentos dentro de `99-archive/` no son fuente oficial.
