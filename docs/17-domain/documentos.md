# Documentos

## Concepto

Un documento representa una entidad de negocio, no solo un archivo.

## Reglas

- Documento lógico y archivo físico son conceptos distintos.
- La metadata no reemplaza columnas oficiales.
- La clave documental la calcula backend.
- Todo documento puede tener varias versiones físicas.
