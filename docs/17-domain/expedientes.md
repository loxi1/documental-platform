# Expedientes

## Concepto

El expediente agrupa documentos relacionados a una operación.

## Contenido esperado

- Documento principal.
- Factura.
- Guía.
- Nota de ingreso.
- Transferencia.
- Detracción.
- Alertas y observaciones.

## Regla

Revisión Contable trabaja por expediente completo.
