# UI Foundation

**Estado:** Aprobado  
**Responsable:** Maestro Sucesor II  

---

## Objetivo

UI Foundation define la base visual y de interacción común de la plataforma.

No es un Design System corporativo completo todavía. Es una base reutilizable para construir producto con consistencia.

---

## Incluye

- layouts
- componentes
- patrones
- estados visuales
- badges
- modales
- tablas
- empty states
- loading states
- navegación
- responsive
- accesibilidad

---

## Regla

El frontend puede orquestar flujos de interfaz, pero no implementar reglas de negocio.

---

## Principio

Misma acción, mismo texto, mismo color y mismo comportamiento en toda la plataforma.
